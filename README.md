# 📊 Netflix Business Case Study - Exploratory Data Analysis

## 📌 Overview

This project performs **Exploratory Data Analysis (EDA)** on Netflix’s dataset to uncover insights related to content trends, genres, and production patterns. The goal is to generate **data-driven recommendations** for business decision-making.

---

## 🎯 Objective

* Analyze Netflix content data to identify trends in **movies vs TV shows**
* Understand **genre distribution, country-wise production, and release patterns**
* Provide insights to support **content strategy and growth decisions**

---

## 📂 Dataset

* Dataset contains information about **8,800+ Netflix titles** including:

  * Title, Type (Movie/TV Show)
  * Director, Cast, Country
  * Release Year, Rating, Duration
  * Genre and Description 

---

## 🛠️ Tech Stack

* **Python**
* **Pandas, NumPy** (Data Processing)
* **Matplotlib, Seaborn** (Visualization)
* **Jupyter Notebook**

---

## 🔍 Key Analysis Performed

* Data Cleaning and Missing Value Handling
* Univariate and Bivariate Analysis
* Trend Analysis (Year-wise content growth)
* Genre and Content Type Distribution
* Country-wise Content Production Analysis

---

## 📈 Key Insights

* Significant growth in **TV Shows after 2015**
* Movies dominate overall content, but TV Shows are increasing rapidly
* Certain countries contribute the majority of content production
* Popular genres and ratings drive platform engagement

---

## 🚀 Results & Impact

* Identified **content trends and audience preferences**
* Provided insights for **content production and release strategies**
* Demonstrated strong skills in **EDA, visualization, and business analysis**
